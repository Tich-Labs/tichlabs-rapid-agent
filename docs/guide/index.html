2. AI Agent — 5-Step Autonomous Workflow

Powered by Gemini 2.5 Flash. The agent reasons through SGBV incidents, plans multi-step responses, and executes tools under human oversight.

1. Assess → 2. Match → 3. Document → 4. Persist → 5. Review

Risk Assessment

Analyzes incident type, description, survivor age/gender → risk score (0-100), severity, urgency, safety actions

Service Matching

Matches against 176 verified services (health, police, shelter, psychosocial, legal, economic empowerment) by location, incident type, needs

FHIR R4 Bundles

Generates interoperable healthcare records (Patient, Observation, ServiceRequest) for Kenya MoH systems

Case Analytics

MongoDB Atlas full-text search + aggregation — surface trends by location, type, severity, or time period

Human-in-the-Loop

All AI recommendations require human approval. Risk assessments include confidence indicators. Anonymous by design.

4. Incident Workflow

New → Assigned → PFA In Progress → Under Review → Escalated → Resolved → Closed

Auto Risk Assessment

Gemini 2.5 Flash evaluates every new incident — risk score, severity, recommended actions

AI Referral Recommendations

Agent matches survivor to nearest verified services. Staff approve or reject with notes.

Workflow Tip

Use the status tabs on the Incidents page to filter cases by stage.
